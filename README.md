**FameStudio Bypass** — это легковесная утилита с графическим интерфейсом для восстановления доступа к **YouTube** и **Discord**. Проект объединяет мощь `winws` и удобство современного UI.

## ✨ Основные возможности
* 🔴 **UI в стиле FameStudio:** Темная тема с красными акцентами.
* ⚡ **Быстрые пресеты:** Отдельные кнопки для обхода ограничений YouTube и Discord.
* 📡 **Live Ping:** Мониторинг задержки сети в реальном времени прямо в интерфейсе.
* 🎮 **Discord Presence:** Твой статус в Discord обновляется автоматически при включении обхода.

## 📦 Состав сборки (v1.3.2)
Для работы программы необходимы следующие файлы, включенные в этот репозиторий:
* `FameStudio_GUI.exe` — исполняемый файл приложения.
* `discord-rpc.dll` — библиотека для интеграции с Discord.
* `glfw3.dll` — библиотека для отрисовки графического интерфейса.
* `bin/` и `lists/` — ядро обхода и списки доменов.

## 🚀 Инструкция по запуску
1.  Перейдите в раздел **[Releases](https://github.com/itzrealman/FameStudio-Bypass/releases)** и скачайте последний архив.
2.  Распакуйте все файлы в одну папку.
3.  Запустите **FameStudio_GUI.exe** от имени администратора.
4.  Выберите нужный режим и наслаждайтесь свободным интернетом.
